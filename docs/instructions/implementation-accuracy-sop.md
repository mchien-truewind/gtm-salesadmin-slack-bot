# Implementation Accuracy SOP

## Steps
1. Extract an acceptance checklist (and explicit must-nots) into the task log before coding.
2. Record assumptions as you make them (what you assumed, why, and what you chose as the default).
3. Implement only what the checklist requires.
4. Produce evidence proportional to risk (tests, logs, screenshots).
5. Run Execution Gate before PR submission and paste the reviewer output into the task log.
